Insertar o Insert


Para insertar datos en mysql desde php luego de hacer la conexión Se crean las variables que recibirán los datos del formulario para este caso Clave, Nombre, Sexo



  • Luego se valida que la consulta se haya ejecutado exitosamente de lo contrario envie un mensaje de error, a su vez al revisar nuestro formulario y la base de datos comprobamos que esta correcto el INSERT

Eliminar o Delete


Para eliminar datos en la BD desde php primero se crea un formulario para de forma visual poder realizar el DELETE ,luego de echa la conexión PHP-MySQL se crea una consulta la cual recibe le parámetro de la clave con la cual eliminara todos los datos que tengan la clave digitada y con el botón eliminar registro se ejecuta esta acción



  • Luego de echa esta acción nos sale un mensaje en pantalla que no s indica que los datos han sido eliminados los cuales al verificar en MySQL se confirma que los datos con clave 1 han sido eliminados
