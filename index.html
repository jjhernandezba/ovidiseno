Información del OVI


Este objeto virtual de información OVI, presenta ante usted contenidos de lectura y multimediales de tipo libre; relacionados con las temática Programación web 1 Vinculación de BD a una página con PHP, este OVI será de mucha ayuda en la ampliación de la información que puede requerir para mostrar la programación y los elementos necesarios para esta vinculación.


Mostrar de manera didáctica e interactiva la forma de programar una web con PHP con conexión a MYSQL y su respectivo CRUD a través de este OVI


Objetivos del Curso


El objetivo de este curso es aprender a diseñar un sitio web con html5 y CSS3, y en esta ocasión lo haremos diseñando un OVI con información de Programación con PHP y MYSQL.


Objetivos especificos


  • Explicar cómo crear una la base de datos en MYSQL
  • Explicar cómo se hace una conexión a la base de datos MYSQL desde PHP
  • Mostrar de manera interactiva como hacer un CRUD con PHP y MYSQL
