Crear una Base de datos con MySQL



Sarmiento(2015).MySQL.Recuperado de http://www.marcossarmiento.com

Para crear una base de datos por de manera grafica con MySQL Worbench damos click en crear nuevo esquema agregamos el nombre elegimos el collation este es la forma como la base de datos interpretara los datos en este caso elegimos UTF8



  • También se puede crear con código de manera sencilla por medio de la sentencia CREATE DATABASE NOMBREBD; Con DROP DATABASE NOMBREBD eliminamos la base de datos creada

  • Crear una tabla CREATE TABLE ALUMNOS_UNAB(ID_ALUMNO INS NOT NULL) o de manera gráfica la cual es más sencilla. Damos clic en nueva tabla le asignamos un nombre, si se desea se le adiciona un comentario y luego procedemos a crear sus atributos y seleccionamos el tipo de dato
